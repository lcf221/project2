Hey Eloise
==========

The site loads Hey Eloise from Adobe Fonts (Typekit) — see the stylesheet link in index.html
(use.typekit.net). No local font files are required for that setup.

Optional: for offline use, you could add self-hosted files and an @font-face in style.css instead.
Title art in assets/outlines/*.svg is loaded as images — it does not use the webfont.
